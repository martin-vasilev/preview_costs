Explanation of the data files:

The data frames used in the analysis are:

- data: containts      all fixation duration data on the target words
- quest_accuracy:      contains the question accuracy data from the experiment
- raw_fix:             contains all fixations made in the experiment (including various saccadic measures)
- t:                   contains trial times
- FX                   contains number of fixations data
- P.txt                contains participant demographic data

All files have an .Rda version (used in R), and .csv version that can be imported in software of your choice
There are also a number of other files in the data, but they were either generated during the pre-processing stages,
or they are not necessary to reproduce the analyses.
